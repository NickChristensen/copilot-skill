# Runtime References

Use this folder for normal, day-to-day API work.

- `copilot-api/copilot-api.operations.yaml`: operation index
- `copilot-api/operations/*.graphql`: canonical query documents
- `copilot-api/examples/requests/*.request.json`: request variable examples
- `copilot-api/enum-values.json`: observed enum candidates for `show` hints
- `operation-groups.md`: operation discovery map by UI area
- `runner-templates.md`: common `copilot-gql` commands
